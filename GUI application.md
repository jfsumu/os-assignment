# Mini Operating System Simulator

A Python-based Mini Operating System Simulator that demonstrates fundamental operating system concepts, including CPU scheduling, memory management, page replacement, synchronization, deadlock handling, and file management.

To see the implementation of the GUI using Tkinter:

## Requirements

- Python 3.13.1

## Running the Application

1. Ensure Python 3.13.1 is installed on your system.
2. Download or clone this repository.
3. Open a terminal in the project directory.
4. Run the GUI application using:

```bash
python gui.py
